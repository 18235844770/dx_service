#### 一、下注与加注

**庄家下家跟注**：要下 **2×皮**。

**第一次加钱/说钱**：至少 **5×屁（最小单位）**。

**后续再加钱**：至少 **等于上次别人加的钱（≥1倍）**；不够就算**敲**（按敲的力度处理）。

------

#### 二、发牌 / 发照 / 比牌节奏

**第一圈（两张牌）可以发照、可以说钱。**

**第二圈发照时：只要有人“拖动”**

- 就表示**有人敲波波了**（等价于第二圈有人顶格动作）。

**第三圈不能发照**

- **直接比牌定胜负**（第三圈就是收尾比牌圈）。

------

#### 三、修忙、皱芒与芒果的产生/断连

**修忙（休芒）成立条件**：

**第一圈有人买牌**，但**第二圈没人说钱** → 也算修忙。

（核心就是：最后没形成有效对抗/没比牌地结束。）

**第二圈有人说钱但没人买牌**：

- **不产生芒果**，
- **尾大吃皮**（最后加钱/说钱那家把皮吃走）。

**忙数（1忙/2忙/3忙）**

- 只在**连续“没人买牌的局”**时累计。
- 中间**一旦比牌**或出现正常买牌对抗 → **芒果断连重计**。
- **到3忙后**：没人说钱时**不再增加也不减少**，一直保持3忙。

**芒果注倍数**

- 1忙：**2×皮**
- 2忙：**4×皮**
- 3忙：**6×皮**

------

#### 四、啵啵（补码）规则

去皮、去芒果后，如果你剩余筹码 **< 最少代入的60%**：

- 必须加啵啵，
- **至少补到最少代入的50%**。

------

#### 五、头大/尾大结算保护

**尾大吃皮**：

- 在“第二圈有人说钱但没人买牌”的局里，尾大（最后说钱/敲的那家）直接吃皮赢走当局。

**头大保命**：

- 头大（牌最大那家）**只输芒果和皮**，不再额外亏别的说钱筹码。
- 等于给“头大”一个止损保护：再差也就交底皮+芒果注。
